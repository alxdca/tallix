# Settings and Preferences

Settings are split into categories, accounts, and preferences.

## Categories

- Create and edit income and expense groups.
- Add items within groups.
- Drag and drop to reorder or move items.

## Accounts

- Manage payment methods and institutions.
- Toggle savings flags and choose savings types.
- Set settlement day and linked accounts.

## Preferences

- Theme (light or dark).
- Decimal separator.
- Toggle whether budget numbers appear below actuals.
- Start year for budgeting history.

Changes to the start year can create missing years automatically.
